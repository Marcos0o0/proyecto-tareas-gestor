# Gestor de Tareas

Aplicación web para gestionar tareas mediante un tablero Kanban con tres estados: Pendientes, En Progreso y Completadas.

## Tecnologías utilizadas

- HTML5
- CSS3 (variables, flexbox, diseño responsive)
- JavaScript vanilla

## Estructura del proyecto

```
proyecto_tareas_gestor/
├── index.html
├── README.md
├── assets/
│   ├── css/
│   │   ├── style.css
│   │   └── responsive.css
│   ├── js/
│   │   └── script.js
│   └── img/
│       └── logo.png
└── docs/
    └── uso_ia.md
```

## Funcionalidades

- Agregar tareas con descripción, prioridad y fecha límite
- Mover tareas entre columnas: Pendientes, En Progreso y Completadas
- Eliminar tareas en cualquier estado
- Contadores automáticos por columna
- Validación de campos obligatorios
- Diseño responsive para móvil, tablet y escritorio

## Cómo usar

1. Clonar o descargar el repositorio
2. Abrir `index.html` en el navegador
3. No requiere instalación ni dependencias externas
