Ajustes manuales realizados

Reflexión crítica
El uso de la IA fue útil para generar rápidamente la estructura base del README, un archivo que por lo general es repetitivo y sigue convenciones estándar. Sin embargo, es importante revisar el contenido generado para asegurarse de que los datos como el nombre del autor, el correo y las funcionalidades descritas sean correctos y correspondan al proyecto real. La IA no tiene acceso directo al código, por lo que podría describir funcionalidades incorrectas si el prompt no es suficientemente específico.
